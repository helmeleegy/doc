Step 1: Sign up for Ponder Account

If you haven‚Äôt already, please go to Ponder‚Äôs product page and sign up for Ponder using your email address.

Step 2: Create Snowflake Account (if you haven‚Äôt already)

If you have a snowflake account you can skip this step. Otherwise, you first need to create a snowflake account. Follow our step-by-step guide on how to create an account on Snowflake.

Step 3: Launch Ponder Studio

After signing in to your account, go to the ‚ÄúWorkspace‚Äù tab and press ‚ÄúLaunch Ponder Studio‚Äù button. This will open a new tab that shows your Jupyter directory.

Step 4: Open a Jupyter notebook

You can either open an existing template notebook called example_db_pushdown.ipynb that exists on your Jupyter directory or open a new notebook. example_db_pushdown.ipynb notebook already contains an template code snippest for connecting to your Snowflake database.

You need to set the notebook kernel to pdkernel. This is under Kernel > Change kernel tab at the top of your notebook page. See figure below for more details.

Step 5: Connect to your DB and start Pondering üéâ

In order to connect to your Snowflake account, you need to enter Snowflake connection information and initialize the connection.

Follow our step-by-step guide on how to find your Snowflake connections information.

import modin.pandas as pd
import ponder.snowflake
from ponder.utils.core import Teleporter

# Create a Ponder Snowflake Connections object
snowflake_con = ponder.snowflake.connect(
    user=*****, #your username that you entered during sign up
    password=*****, #password that you entered during sign up
    account=*****, #You can find this under ‚ÄúAdmin‚Äù > ‚ÄúAccounts‚Äù > "Locator" field (You can also find your account in the URL: https://app.snowflake.com/us-west-2/<account-name>/organization, put the <account-name>, not including the URL
    role=*****, # your access control level. By default this is ‚ÄúACCOUNTADMIN‚Äù.
    database=*****, # database = ‚ÄúTEST‚Äù or whatever you named the database in Step 1
    schema=*****,   # schema= ‚ÄúPUBLIC‚Äù,
    warehouse=***** # warehouse= ‚ÄúCOMPUTE_WH‚Äù
)

# Initialize the Snowflake connection
ponder.snowflake.init(snowflake_con, timeout=1200)

After connecting to your database, you can read a table that exists on your Snowflake account using following command.
